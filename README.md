# conversor-api
Api de consumo de serviços de conversão.

Desenvolvido com Spring Boot 

Requisitos de compilação:
1.Java JDK 8 ou superior
2.Maven 3

Compilação
    mvn clean package

Subir Aplicação
    mvn spring-boot:run

Testes - JUnit
    mvn test
    
EndPoint 
    Executar a url 
    Ex: Via Browser ou curl http://localhost:8080/api/conversor-numeral/0
